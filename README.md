# InternShield 🛡️

**InternShield** is a modern security platform designed to protect students from internship scams. By analyzing key indicators of an internship opportunity, InternShield helps users identify whether an offer is **Genuine**, **Suspicious**, or **Fake** before they invest time or money.

---

## 🚀 The Mission
Internship scams are becoming increasingly common, targeting students who are eager to start their careers. InternShield provides a simple, automated way to verify opportunities and flag common red flags.

## ✨ Core Features
- **Internship Verification:** Enter company details, email addresses, stipends, and websites to get an instant risk assessment.
- **Scam Detection Logic:**
    - 🚩 **Monetary Demands:** Flags any internship that asks for upfront "training" or "security" fees.
    - 📧 **Domain Analysis:** Detects suspicious email domains (e.g., using personal Gmail/Yahoo accounts instead of official company domains).
    - 🌐 **Web Verification:** Checks for the existence and legitimacy of official company websites.
    - 💰 **Unrealistic Offers:** Identifies offers that seem too good to be true based on industry standards.
- **Result Indicators:**
    - ✅ **Genuine:** Verified and trustworthy.
    - ⚠️ **Suspicious:** Potential red flags found; proceed with caution.
    - ❌ **Fake:** Confirmed scam; do not interact.

## 🔧 Installation & Running
1. Clone the repository:
   ```bash
   git clone https://github.com/gayatrishete619/internshield.git
   ```
2. Navigate to the project directory:
   ```bash
   cd internshield
   ```
3. Compile the Java files:
   ```bash
   javac ModernLoginPage.java DashboardPage.java
   ```
4. Run the application:
   ```bash
   java ModernLoginPage
   ```
